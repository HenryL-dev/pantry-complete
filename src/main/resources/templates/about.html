About The Pantry

The Pantry is a Recipe Search/ Meal Planning WebApp that displays recipes determined by what ingredients the user has available. The user may also create an account that allows them to submit their own/favorite other recipes, and all their data is stored in a database with BCrypt password encryption.

The Pantry Development & Testing

This site was developed as a Liftoff group portfolio project to display knowledge learned from the LaunchCode LC101 course.
