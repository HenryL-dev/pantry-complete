Features and Functionality

End User
  • Browse recipes (list recipe cards w/ link to recipe)
    • Pagination of recipes
  • View full recipe (display recipe)
  • Simple search (return recipe cards matching search term)
  • Advanced search (return table of recipes matching ingredient search terms)
  • Random recipe (return a random recipe)
Administration
  • Add Units (add, edit, delete)
  • Add Ingredients (add, edit, delete)
  • Add Recipe (add, edit, delete)
    • Name
    • Description
    • Instructions
    • Image (upload and delete)
    • Recipe ingredients (add, edit, delete)
      • Amount
      • Unit
      • Ingredient
    • Add new ingredient (for availability)

Extensive error checking for admin included for adding, editing and deleting. Including modal popup warnings and error/success messages.
